Privacybeleid

Laatst bijgewerkt: maart 2026

Introductie

Ik ga zorgvuldig om met je gegevens. Op deze pagina lees je precies wat ik bewaar, waarom, en wat je rechten zijn.

Wat ik verzamel via het contactformulier

Als je het contactformulier op mijn site invult, worden de volgende gegevens verzameld:

  • Je naam
  • Je e-mailadres
  • Type project
  • Voorkeursdatum
  • Je bericht

Technische gegevens

Daarnaast worden automatisch enkele technische gegevens opgeslagen:

  • IP-adres
  • Browser-informatie (user agent)
  • De pagina van waaruit je het formulier hebt verstuurd (referrer)

Waarvoor ik je gegevens gebruik

Ik gebruik je gegevens alleen om je aanvraag te beantwoorden. Je bericht wordt doorgestuurd naar info@byhava.nl zodat ik je zo snel mogelijk kan helpen.

Cookies en tracking

Deze website gebruikt geen cookies, geen analytics en geen tracking scripts. Niks. Nada.

Hoe lang ik je gegevens bewaar

Inzendingen via het contactformulier worden maximaal 12 maanden bewaard. Wil je dat je gegevens eerder worden verwijderd? Stuur me een mailtje en ik regel het.

Je rechten

Onder de AVG (GDPR) heb je het recht om:

  • Je gegevens in te zien
  • Je gegevens te laten corrigeren
  • Je gegevens te laten verwijderen

Neem hiervoor contact op via info@byhava.nl.

Verwerking door derden

Het contactformulier wordt verwerkt via Formbridge (forms.bollenstreekdigitaal.nl). Dit is een formulierverwerker die de inzendingen veilig opslaat en doorstuurt.

Wijzigingen

Ik kan dit privacybeleid van tijd tot tijd aanpassen. De laatste versie vind je altijd op deze pagina.

Vragen?

Heb je vragen over je privacy? Mail me op info@byhava.nl.
